# Simple line counter

## Currently supported languages
- TypeScript (.ts | .tsx)
- JavaScript (.js | .jsx)

## Install
```npm install -g manylines```

## Usage
``` $ manylines [ DIRECTORY ] ```
### Flags
- --files   |   shows all files in a directory with their number of lines
